Educational Quiz Generator - API Documentation

Online Document

Welcome to the API documentation for the Educational Quiz Generator. This system is designed to recognize objects in images using YOLO and generate quiz questions using ibm-granite/granite-3.1-2b-instruct. Below is a detailed explanation of the available endpoints.

1. Recognition Endpoint (`/recognize/`)

The POST /recognize/ endpoint allows you to upload an image for object detection. The system processes the image and returns a detected objects.

Request Type: POST

Parameters:

  • file (required): The image file to be uploaded (supported formats: JPG, PNG).

Example Response:

{
    "message": "success",
    "data": "cell phone"
}

2. Quiz Generation Endpoint (`/generate/`)

The GET /generate/ endpoint generates quiz questions. The system allows to create questions tailored to the detected objects.

Request Type: GET

Parameters:

  • topic (required): The topic of the quiz (e.g., "History", "English").
  • number (required): The number of quiz questions to generate.
  • ageGroup (required): The age group for which the quiz is intended (e.g., "12-15").
  • item (optional): The item you want to generate quiz questions around

Example Response:

{
    "message": "success",
    "data": [
        {
            "question_text": "Question: Which of the following is NOT a way a business can generate revenue?",
            "choiceA": "A) Selling products",
            "choiceB": "B) Providing services",
            "choiceC": "C) Donating to charity",
            "choiceD": "D) Giving away free samples",
            "answer": "Answer: C) Donating to charity"
        }
    ]
}
